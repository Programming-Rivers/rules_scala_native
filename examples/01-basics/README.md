This section covers the fundamental concepts of setting up and building Scala Native projects with Bazel.

- **[00-bazel-setup](00-bazel-setup)**: Learn how to configure your `MODULE.bazel` and `.bazelrc` for Scala Native.

- **[02-transitive-dependencies](02-transitive-dependencies)**: How to work with external libraries and manage complex dependency graphs.
- **[03-testing](03-testing)**: Using JUnit to test your native code within the Bazel environment.
- **[05-static-and-dynamic-libraries](05-static-and-dynamic-libraries)**: Understanding how to create and link against different library types.
